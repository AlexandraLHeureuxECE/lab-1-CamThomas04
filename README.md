# Tic-Tac-Toe (Web) Created with Codex Powered by ChatGPT 5.2

Simple two-player Tic-Tac-Toe playable in a browser (same device).

## Run

Serve locally:
  - `python3 -m http.server`
  - visit `http://localhost:8000`

Or GitHub Pages URL: https://alexandralheureuxece.github.io/lab-1-CamThomas04/

## Controls

- Mouse/touch: click a square to place your mark.
- Keyboard:
  - Arrow keys: move around the board
  - Enter / Space: place your mark (on the focused square)
  - `1`â€“`9`: jump focus (top-left = `1`, bottom-right = `9`)
  - `R`: restart at any time
  - `T`: toggle light/dark theme
