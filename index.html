Tic‑Tac‑Toe

Two players, one device.

Instructions

Click/tap a square (or use the keyboard below). Get 3 in a row to win.

Keyboard

  • Arrow keys: move focus
  • Enter / Space: place mark
  • 1–9: place directly (1 = top-left)
  • R: restart game
  • T: toggle light/dark theme
  • Esc: close this dialog
